Direktorij algortihm_detailed sadržava datoteke ship_port_algorithm.py i gale_shapley_algorithm.py čiji output osim rezultata, 
sadržavaju i popratne informacije o njihovom konstruiranju. Outputi su priloženi u .txt datotekama, čija su imena naznačena 
u komentarima datoteka ship_port_algorithm.py i gale_shapley_algorithm.py prije zadavanja problema.
Pozivi metoda za probleme su zakomentirani.

Ako pokrenete programe samostalno, konačni rezultati će odgovarati onima u priloženim output .txt datotekama, ali ne nužno i 
popratni ispisi jer je rječnik neuređena kolekcija i jer se u Gale-Shapleyjevom algoritmu slobodan muškarac (odnosno brod),
bira proizvoljno.

Za ispis koji sadrži samo rješenje, bez popratnog ispisa, pogledajte direktorij algorithm koji sadrži istoimene datoteke u kojima je
implementiran Ship-Port algoritam, kao i istoimene datoteke s outputima.

**

Directory algortihm_detaied contains files ship_port_algorithm.py and gale_shapley_algorithm.py, whose outputs, besides the result,
contain information about the construction of the result. Outputs are available in .txt files, whohe names have been marked in 
files ship_port_algorithm.py i gale_shapley_algorithm.py before definitions of the problems.
Lines of code for the execution have been commented.

After running the programs, final results will correspond to those in the output .txt files, but extra outputs might be different. There
are two reasons: dictionary is unordered collection and free man (or in this case, boat) is chosen randomly in Gale-Shapley algorithm.

For the outputs which contain results only, without extra output about creating the result, check the directory algorithm which 
contains files of the implementation of the Ship-Port algorithm and results also, equally named.
